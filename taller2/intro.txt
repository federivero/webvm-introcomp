
Bienvenidos al taller 2 de Introducción a la Computación. En este taller se trabajará sobre la consola de comandos linux y sobre el sistema de archivos. 

El laboratorio está estructurado en varias salas. De cada sala se obtendrá un código que se deberá ingresar en el EVA para aprobar.

Para aprobar el laboratorio se precisará utilizar la consola de comandos, que es a lo único que se tiene acceso en este ambiente.

> comando <parámetro 1> <parámetro 2> ...

Típicamente los parámetros son opcionales, pero algunos comandos tienen parámetros requeridos, como se verá más adelante.

En todo momento se puede ejecutar el comando 'man' para obtener información sobre un comando. De la siguiente forma:

> man <nombre comando>

Los primeros parámetros a utilizar serán CD (cambiar directorio) y LS (listar archivos del directorio actual). Con esos comandos se puede navegar en el sistema de archivos.

------------------

El comando cd permite cambiar de carpeta o directorio. Cuando se ejecuta cd seguido del nombre de un directorio, la terminal pasa a ubicarse dentro de ese directorio. Por ejemplo:

cd sala1

mueve la terminal hacia el directorio sala1, siempre que exista dentro del directorio actual. También es posible volver al directorio anterior utilizando cd .., donde .. representa el directorio padre (el directorio que contiene al actual).

------------------

El comando ls (list) permite listar el contenido del directorio en el que se encuentra la terminal en ese momento, es decir, ver los archivos y directorios dentro de un directorio (por defecto se utiliza el directorio actual). Esto incluye archivos y subdirectorios. Por ejemplo:

ls

muestra el contenido del directorio actual. Este comando es útil para saber qué archivos hay disponibles y a qué lugares se puede navegar.

------------------

El comando cat permite mostrar el contenido de archivos de texto en la terminal. Se utiliza indicando el nombre del archivo que se quiere leer. Por ejemplo:

cat intro.txt

muestra el contenido del archivo intro.txt en pantalla (probablemente es lo que están ejecutando ahora!).

------------------

Con estas direcciones, ya están en condiciones de acceder a la primer sala y leer las instrucciones. Ubicados en la sala principal, ejecutar:

cd sala1
ls
cat instrucciones.txt
